VisionFX AI is an AI-powered Android application designed to provide advanced image enhancement, visual effects, and computer vision capabilities. The application integrates Python backend services using FastAPI and OpenCV with an Android frontend developed in Android Studio.

Features:
• AI-powered image processing
• Real-time visual effects
• Image enhancement and filtering
• FastAPI backend integration
• OpenCV computer vision processing
• Android-friendly user interface

Technologies Used:
• Java
• Python
• FastAPI
• OpenCV
• Android Studio
• REST APIs
• Git & GitHub
